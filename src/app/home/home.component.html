Introduction

Cette plateforme a été conçue pour faciliter les échanges sur des enjeux touchant les Québécois.

Elle est interactive de plusieurs manières :

  • Le menu de gauche permet d'accèder aux sections de la page,
    il peut être fermer avec l'icône     menu

  • Cliquez sur les icônes dans la barre du haut :

    • Pour voir la liste des personnes recent_actors
    • Pour voir la liste des enjeux art_track

Schéma sous-jaçent le site web

Trois principaux éléments vont vous permettre d'explorer les enjeux,
leurs opinions et les personnes qui les ont émisent

Ils sont listé dans le shéma suivant :

  • Un enjeu à :

    • Des opinions qui sont relié à des

    • Des personnes. Chaque Personnes à

      • Une liste d'opinions et

      • Une liste d'enjeu qu'il a créé ou pour lesquels il a manifesté son point de vue (ajouté une opinion, aimé ou pas)
